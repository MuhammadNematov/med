# Медицинский центр «ГМЛ/ГМЦ» — Адаптивный лендинг

Этот проект представляет собой одностраничный сайт для медицинского центра. Он был разработан по макету из Figma и включает в себя все ключевые блоки: шапку с контактами, информацию об услугах и диагностике, популярные направления и важные юридические уведомления. Основная задача — представить информацию о центре в понятном и удобном виде на любых устройствах.

## 🛠️ Технологии

Проект построен на следующих технологиях:

*   **HTML5** — семантическая разметка страницы.
*   **Tailwind CSS** — утилитарный CSS-фреймворк для быстрой стилизации и обеспечения адаптивности.
*   **JavaScript (Vanilla)** — используется для минимальной интерактивности (например, открытие/закрытие мобильного меню).
*   **Font Awesome** — набор иконок для визуального оформления.

📱 Адаптивность
Сайт полностью адаптивен и корректно отображается на всех типах устройств:
